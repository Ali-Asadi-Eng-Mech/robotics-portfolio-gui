# راهنمای ویرایش بدون تغییر کد

این نسخه یک بخش جدید به نام «ویرایش و ذخیره» دارد. از این بخش می‌توانید کارت‌های پروژه را بدون تغییر فایل‌های کد اصلاح کنید.

## قابلیت‌ها

- ویرایش عنوان پروژه، مسئول، ددلاین، اولویت، وضعیت، خروجی، اقدام بعدی، ریسک و چک‌لیست
- افزودن کارت جدید
- کپی کارت موجود
- حذف کارت
- ذخیره خودکار محتوای ویرایش‌شده در مرورگر
- دریافت پشتیبان کامل JSON
- ورود پشتیبان JSON روی دستگاه دیگر
- بازگشت به نسخه اولیه

## محدودیت مهم

این نسخه هنوز دیتابیس مرکزی ندارد. بنابراین تغییرات هر کاربر در مرورگر همان کاربر ذخیره می‌شود. برای اینکه همه کاربران یک نسخه مشترک و آنلاین ببینند، باید نسخه بعدی به دیتابیس مثل Supabase یا Firebase وصل شود.

## روش انتقال تغییرات به کامپیوتر دیگر

1. در اپ وارد بخش «ویرایش و ذخیره» شوید.
2. روی «دانلود پشتیبان کامل» بزنید.
3. فایل `robotics-portfolio-full-backup.json` را به دستگاه دیگر منتقل کنید.
4. در دستگاه مقصد، روی «ورود پشتیبان» بزنید و فایل را انتخاب کنید.

## انتشار روی GitHub Pages

اگر فقط فایل‌های این نسخه را روی GitHub Pages جایگزین کنید، قابلیت ویرایش در سایت فعال می‌شود. دقت کنید فایل‌های زیر باید در ریشه Repository باشند:

- `index.html`
- `styles.css`
- `app.js`
- `data.js`
- `manifest.webmanifest`
- `sw.js`
- `favicon.svg`
- `404.html`
- `.nojekyll`
