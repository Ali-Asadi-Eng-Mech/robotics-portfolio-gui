# قفل رمز برای داشبورد پورتفولیوی رباتیک

این نسخه دارای صفحه ورود است. رمز پیش‌فرض:


## هشدار مهم

این قفل روی GitHub Pages به‌صورت سمت مرورگر اجرا می‌شود. بنابراین برای جلوگیری از ورود کاربران عادی مناسب است، اما امنیت سازمانی کامل نیست؛ چون GitHub Pages سایت را به‌صورت فایل‌های استاتیک منتشر می‌کند و داده‌ها و کدهای JavaScript همچنان از نظر فنی قابل دریافت هستند.

اگر اطلاعات واقعاً محرمانه است، باید از یکی از مسیرهای زیر استفاده شود:

1. Netlify Password Protection روی پلن Pro یا بالاتر.
2. Cloudflare Access با دامنه اختصاصی.
3. نسخه دارای Login واقعی و دیتابیس/بک‌اند.

## تغییر رمز

برای تغییر رمز در همین نسخه، باید SHA-256 رمز جدید را محاسبه و مقدار `passwordHash` را در ابتدای فایل `app.js` جایگزین کنید. نمونه محاسبه در Python:

```bash
python - <<'PY'
import hashlib
password = 'رمز-جدید-اینجا'
print(hashlib.sha256(password.encode()).hexdigest())
PY
```

سپس مقدار خروجی را در فایل `app.js` در این بخش قرار دهید:

```js
passwordHash: '...'
```

پس از Commit در GitHub، سایت با همان لینک قبلی به‌روزرسانی می‌شود.
